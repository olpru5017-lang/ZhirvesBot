# Как обновить бота на Railway

## Проблема
Бот показывает ошибку "davey library needed in order to use voice" - это означает, что PyNaCl не установлен правильно.

## Решение

### Вариант 1: Через GitHub (рекомендуется)

1. Убедись, что проект подключен к GitHub
2. Сделай commit и push изменений:
   ```bash
   git add .
   git commit -m "Fix PyNaCl installation"
   git push
   ```
3. Railway автоматически пересоберёт проект

### Вариант 2: Через Railway CLI

1. Установи Railway CLI:
   ```bash
   npm install -g @railway/cli
   ```

2. Войди в аккаунт:
   ```bash
   railway login
   ```

3. Подключись к проекту:
   ```bash
   railway link
   ```

4. Задеплой изменения:
   ```bash
   railway up
   ```

### Вариант 3: Пересоздать сервис на Railway

1. Зайди на railway.app
2. Открой свой проект
3. Удали текущий сервис
4. Создай новый сервис из GitHub репозитория
5. Railway автоматически соберёт проект с правильными зависимостями

## Проверка

После обновления проверь логи на Railway:
- Должна исчезнуть строка "davey is not installed"
- Бот должен успешно подключаться к голосовым каналам

## Если ничего не помогает

Проблема может быть в том, что Railway кэширует старый образ. Попробуй:

1. Зайди в настройки сервиса на Railway
2. Найди раздел "Settings"
3. Нажми "Clear Build Cache"
4. Нажми "Redeploy"

## Альтернатива: Использовать другой хостинг

Если Railway не работает, попробуй:
- **Render.com** - есть бесплатный план, хорошо работает с Docker
- **Heroku** - классический вариант для ботов
- **VPS** (DigitalOcean, Linode) - полный контроль, но платно

Для Render.com уже есть готовый файл `render.yaml` в проекте!
